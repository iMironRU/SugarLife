# SugarLife / СладкаяЖизнь

Единое мобильное приложение для людей с диабетом 1 типа.

Объединяет в одном опыте мониторинг глюкозы, ведение терапии, расчёт болюсов, обучение через игру и автоматическое управление инсулинотерапией. Заменяет связку из 4–5 разных приложений, которыми вынужденно пользуется современный диабетик.

## Статус

Ранняя стадия. Сейчас прорабатываются концепция и UX-прототипы экранов в HTML до выбора нативного стека.

## Документация

- [`docs/concept.md`](docs/concept.md) — концепция: что это, чем не является, принципы, архитектура
- [`docs/decisions/`](docs/decisions/) — журнал архитектурных решений с обоснованиями

## Прототипы

UX-прототипы доступны через GitHub Pages (включается в Settings → Pages → Source: GitHub Actions).

Локально:

```bash
cd prototypes
python3 -m http.server 8000
# открыть http://localhost:8000
```

## Структура

```
.
├── docs/             Концепция и журнал решений
├── prototypes/       HTML-прототипы интерфейса
│   ├── index.html    Главная страница со списком экранов
│   ├── styles.css    Дизайн-токены (цвета, отступы)
│   └── screens/      Отдельные экраны
└── .github/          GitHub Actions для авто-деплоя на Pages
```
